# 프로젝트 명
Null Crowd Funding

## 프로젝트 소개
크라우드 펀딩 플랫폼을 등록하고 후원 및 문의 등 간편히 사용할 수 있는 크라우드 펀딩 웹 페이지 입니다.

## 프로젝트 작업 기간
2023.12.05 ~ 2023.01.13

## 프로젝트 참여 인원
5명

## 기획 사유
길어진 코로나 시기에 지친 사람들에게 누구든지 특별한 공연, 페스티벌, 공간, 이벤트 등을 기획하고, 참여하며
색다른 문화를 즐기고, 다양한 문화를 담을 수 있는 공간을 제공하고자 크라우드 펀딩 프로젝트를 기획하게 되었습니다.
